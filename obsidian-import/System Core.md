# System Core

Este arquivo documenta a fonte unica de logica do sistema.

O codigo real fica em:

`00 - System/System Core.js`

Os dashboards em `04 - Stats` carregam esse arquivo para evitar formulas duplicadas.

## Fonte dos dados

- Daily Notes em `01 - Daily`
- Tarefas concluidas com `- [x]`
- Tags e nomes das missoes
- Datas pelo frontmatter `date` ou nome `YYYY-MM-DD`

## Metricas centralizadas

- XP diario
- Penalidades
- Descanso estrategico
- Dia valido
- Dia perfeito
- Score de desempenho
- Taxa de missao por habito
- Periodos semanais, mensais e anuais
